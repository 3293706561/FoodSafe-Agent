## About 区短介绍

```text
FoodSafe Agent / 食安智问 是一个面向食品企业标签审版与法规咨询的作品集项目。它不依赖外部 API，通过结构化法规条款、轻量检索和显式规则，将食品标签问题转成风险等级、整改建议和可追溯引用依据；对知识库外问题返回人工核验，避免无依据法规结论。
```

## 置顶项目说明

```text
食品科学背景 + AI 应用 + 企业合规场景的作品集项目。当前版本覆盖 30 条结构化法规条款和 30 条黄金测试问题，提供 CLI 与静态 Web Demo，可部署到 GitHub Pages。
```
