# 路线图

## M0：离线闭环（已完成）

- JSON 样例法规库、轻量检索、风险规则、整改建议、来源引用与拒答。
- 3 个自动化验收场景；Python 标准库直接运行。

## M1：作品集演示版

- 增加静态 Web Demo、标签文本粘贴区和结构化合规报告展示，优先适配 GitHub Pages，避免免费服务休眠影响演示。
- 扩充 GB 7718、GB 28050、GB 2760 的经授权/公开语料与 30 条黄金测试集。
- 为条款加入发布日期、生效日期、失效日期、产品类别和版本关系。

验收：Top-3 检索命中率、引用正确率和拒答准确率均可复现；演示问题一键运行。

## M2：RAG + LLM

- BM25 与中文向量混合检索，加入 reranker。
- 通过环境变量接入可替换 LLM；生成只能使用检索上下文。
- 增加引用一致性校验、冲突条款提示、人工复核队列和审计日志。

验收：答案中的每个合规结论都能映射到有效条款；无依据问题必须拒答。

## M3：企业场景

- 标签 OCR、配料/添加剂实体抽取、产品类别路由、法规更新流水线。
- 企业内部标准与国家标准对照、整改工单、权限控制和版本回放。
